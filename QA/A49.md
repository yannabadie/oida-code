## Decision

Proceed with the offline static audit as Phase 6.a / ADR-68.

This closes only the static consistency lane of G-6a. A passing audit
means "no static inconsistency detected" across the archived replay
files. It does not prove provider-independent replay validity, upstream
PR truth, product safety, or semantic correctness.
