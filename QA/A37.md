Format obligatoire pour toute demande cgpro :

CGPRO_REQUEST
- title:
- phase:
- case_id:
- task_type:
- why_human_required:
- inputs_to_review:
- exact_decision_needed:
- allowed_answers:
- files_to_update_after_response:
- blocking: true|false
- deadline_or_priority:
END_CGPRO_REQUEST

Exemple pour un label opérateur :

CGPRO_REQUEST
- title: Human label required for operator soak case_001
- phase: 5.8
- case_id: case_001_oida_code_self
- task_type: human_label
- why_human_required: The project rules forbid Claude from writing operator labels or UX scores. The label must reflect a human judgment after reading the GitHub Step Summary, summary.md, grounded_report.json and audit logs.
- inputs_to_review:
  - operator_soak_cases/case_001_oida_code_self/README.md
  - .oida/operator-soak/case_001/summary.md
  - .oida/operator-soak/case_001/grounded_report.json
  - .oida/operator-soak/case_001/audit/
- exact_decision_needed: Choose one operator_label and write a 3-10 line rationale.
- allowed_answers:
  - useful_true_positive
  - useful_true_negative
  - false_positive
  - false_negative
  - unclear
  - insufficient_fixture
- files_to_update_after_response:
  - operator_soak_cases/case_001_oida_code_self/label.json
- blocking: true
- deadline_or_priority: high
END_CGPRO_REQUEST

Exemple pour un score UX :

CGPRO_REQUEST
- title: Human UX score required for operator soak case_001
- phase: 5.8
- case_id: case_001_oida_code_self
- task_type: ux_score
- why_human_required: UX usefulness cannot be inferred automatically. It must be judged by a human operator.
- inputs_to_review:
  - GitHub Step Summary for the run
  - .oida/operator-soak/case_001/summary.md
  - .oida/operator-soak/case_001/grounded_report.json
- exact_decision_needed: Score readability, traceability, actionability and absence of false verdict.
- allowed_answers:
  - summary_readability: 0|1|2
  - evidence_traceability: 0|1|2
  - actionability: 0|1|2
  - no_false_verdict: 0|1|2
  - notes: string
- files_to_update_after_response:
  - operator_soak_cases/case_001_oida_code_self/ux_score.json
- blocking: true
- deadline_or_priority: high
END_CGPRO_REQUEST
